## NYC OpenData 311 Service requests Dashboard

## Developing ML models capable of distinguishing European Portuguese from Brazilian Portuguese
#### Program: Python (Packages: Pandas, Pytorch, Sklearn)
#### Authors: José Pedro Rodrigues, Pedro Leite Ferreira, Sandra Silva
From an initial LSTM RNN model with an output layer with two neurons (classification in PT-PT or PT-BZ) explored numerous experimentation tests through modifying its hyperparameters (No. Epochs, Activation functions, Learning Rates, Batch Number, Optimization function, and No. of hidden layers and neurons), and using Pre-trained components (Word embeddings and Pre-trained models). The best performing model consisted of using the pre-trainined model Bertimbau which had an F1-test score of 98%.

## Kidney Exchange Program Improvement
#### Program: AMPL and Python
#### Authors: José Pedro Rodrigues
Prediction model development of Kidney exchange program with the purpose of finding the best donor-patient match that improves the maximum survival time for patients possible. Included further variables that have an effect on survival time (Smoker, weight, human leukocyte antigens B) and a rationality constraint  (Pair will only accept cross transplant if survival time is higher than internal transplant)

## Deepdive into the assumptions and characteristics of ML methods
#### Program: Python (Packages: Pandas, Numpy, Matplotlib, Scikit-learn and Seaborn)
#### Authors: José Pedro Rodrigues, Pedro Leite Ferreira, Sandra Silva
Generated several artificial datasets with different conditions and explored various Machine learning method assumptions, Bias-Variance trade-offs, and different Ensemble Techniques (Bagging, Random Forest, AdaBoost). ML methods explored include Logistic Regression, Linear Discriminant Analysis, Quadratic Discriminant Analysis, Decision Tree, and Support Vector Machines.

## Phylogenetics of the hemoglobin gene

#### Authors: José Pedro Rodrigues, Cecília Santos, Tomás Ferreira
##### _(Private Repository)_

## Analysis and forecast of the Earth Surface temperature

#### Authors: José Pedro Rodrigues

## Evaluating the Prediction Accuracy of an FPL Player Points Using Nonparametric and Parametric Methods
#### Program: R
#### Authors: José Pedro Rodrigues
