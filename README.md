# Projects 2023 - 2025
Compilation of academic and non-academic projects developed between 2023 and 2025 in data science, machine learning, and optimization, implemented using multiple programming languages.
## Modeling a Quality-Oriented Evaluation System for Primary Health Care Teams in Portugal
#### Program: Python and R
#### Author: José Pedro Rodrigues
##### _(Currently in the process of publication)_
Evaluated the current Portuguese commissioning system (IDE), and developed an alternative framework to determine the true quality of Primary Health Care units in Portugal through a more personalised and mathematically accurate approach.
## NYC OpenData 311 Service requests Dashboard
#### Program: R (Libraries: Shiny)
#### Authors: José Pedro Rodrigues, Diana Simôes, Tomás Ferreira, Ana Luísa Rocha
Developed a dashboard to analyze complaint patterns, identify hotspots, and evaluate the effectiveness of service responses. Multiple hypotheses were evaluated, including but not limited to: the effect of weather conditions on the number of complaints; how the complexity of requests affects response time; and the frequency of noise complaints on lively nights.

## Statistical and Analytical study of a Cars Sales dataset 
#### Program: R and SPSS
#### Authors: José Pedro Rodrigues, Pedro Leite
The dataset was obtained from Kaggle and analyzed using univariate, bivariate, and multivariate techniques to explore different statistical methods. Multivariate methodologies included PCA, factor analysis, cluster analysis, LDA, and linear regression.

## Developing ML models capable of distinguishing European Portuguese from Brazilian Portuguese
#### Program: Python (Packages: Pandas, Pytorch, Sklearn)
#### Authors: José Pedro Rodrigues, Pedro Leite Ferreira, Sandra Silva
Starting from an initial LSTM RNN model with an output layer of two neurons (for classification into PT-PT or PT-BR), numerous experiments were conducted by modifying its hyperparameters (number of epochs, activation functions, learning rates, batch size, optimization function, and number of hidden layers and neurons), as well as by incorporating pre-trained components (word embeddings and pre-trained models). The best-performing model used the pre-trained BERTimbau model, achieving an F1 test score of 98%.

## Develop Predictive Models for Churn Prediction in a Telecom Company
#### Program: Python (Packages: Pandas, Pytorch, Sklearn)
#### Authors: José Pedro Rodrigues, Pedro Leite Ferreira, Sandra Silva
Analyzed and processed data to develop predictive models for churn prediction. Evaluated multiple models, including Decision Trees, K-Nearest Neighbors (KNN), Naive Bayes, Random Forest, Support Vector Machines (SVM), Neural Networks, and Logistic Regression

## Kidney Exchange Program Improvement
#### Program: AMPL and Python
#### Authors: José Pedro Rodrigues
Developed a predictive model for a kidney exchange program to identify optimal donor–patient matches that maximize patient survival time. Additional variables affecting survival time were incorporated (e.g., smoking status, weight, and human leukocyte antigen B). A rationality constraint was also included, ensuring that a pair would only accept a cross-transplant if the expected survival time exceeded that of an internal transplant.

## Deepdive into the assumptions and characteristics of ML methods
#### Program: Python (Packages: Pandas, Numpy, Matplotlib, Scikit-learn and Seaborn)
#### Authors: José Pedro Rodrigues, Pedro Leite Ferreira, Sandra Silva
Generated several artificial datasets under different conditions and explored various machine learning method assumptions, bias–variance trade-offs, and ensemble techniques (bagging, random forest, AdaBoost). The methods examined included logistic regression, linear discriminant analysis, quadratic discriminant analysis, decision trees, and support vector machines.

## Identify set of Open Reading Frames (ORFs) located in the yeast genome
#### Program: Python
#### Authors: José Pedro Rodrigues, Cecília Santos, Tomás Ferreira
##### _(Private Repository)_
Collected data from the UCSC Genome Browser and developed functions to analyze genome statistics, including nucleotide frequency, codon frequency, and GC content. Identified all potential open reading frames (ORFs) in both positive and negative strands.

## Protein Sequence Classification
#### Program: Python (Packages: Pandas and Scikit-learn))
#### Authors: José Pedro Rodrigues, Cecília Santos, Tomás Ferreira
##### _(Private Repository)_
Conducted an exploratory project to evaluate the feasibility of classifying protein sequences based on dinucleotide composition. Analyzed two protein families (zinc finger and globins) and applied the Feature Frequency Profiles (FFP) approach to formulate a classification problem, exploring multiple machine learning algorithms.

## Phylogenetics of the hemoglobin gene
#### Program: Python (Packages: BioPython and Uniprot)
#### Authors: José Pedro Rodrigues, Cecília Santos, Tomás Ferreira
##### _(Private Repository)_
Explored the phylogenetic evolution of hemoglobin across different species in relation to the human hemoglobin gene (HBB_HUMAN, UniProt P68871). The project included data collection, BLAST analysis, multiple sequence alignment, and the construction of a phylogenetic tree.

## Analysis and forecast of the Earth Surface temperature
#### Program: Python (Packages: Pandas, Numpy, Matplot, Seaborn, Statsmodels and Scikit-learn)
#### Authors: José Pedro Rodrigues
Conducted data analysis on a climate change dataset (Earth Surface Temperature time series) obtained from Kaggle. Applied decomposition methods and stationarity tests to understand seasonality and trends in the data. Forecasting approaches were also explored using SARIMA models, exponential smoothing, and cross-validation.

## Evaluating the Prediction Accuracy of an FPL Player Points Using Nonparametric and Parametric Methods
#### Program: R
#### Authors: José Pedro Rodrigues
Evaluated the accuracy of predictions for Fantasy Premier League (FPL) player points based on performance data. Nonparametric evaluation was conducted using cross-validation and train–test splits, while parametric evaluation was performed using Mallows’ Cp.

## Surgeon Matching Problem
